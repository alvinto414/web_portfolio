Project Description

Create a Parallel coordinates + Scatterplot visualization that reads data from a comma-separated text file. Like last time, each parallel coordinate takes vertical "stripe" as the axis that holds an individual data variable. Each stripe is the same height, with the minimum data value at the bottom, and the maximum at the top. To start with, all variables are shown. Each stripe is of equal width.

Combine this with a scatterplotter that assigns data variables to X, Y, Color, etc. The key feature is brushing between views. That is, selecting data in one view highlights the same data in the other view. This means that individual data points and groups of data points (such as all values in a particular range) must be selectable.
